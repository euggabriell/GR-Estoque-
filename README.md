# GR Estoque

Sistema web de controle de estoque desenvolvido para gerenciamento interno de materiais hospitalares.

## Funcionalidades

- Controle de estoque
- Entrada de materiais
- Saída de materiais
- Histórico completo
- Controle por unidade
- Login com Firebase Authentication
- Firestore Database
- Dashboard com métricas
- Cadastro de usuários

## Tecnologias

- HTML5
- CSS3
- JavaScript
- Firebase Authentication
- Cloud Firestore
- Bootstrap 5

## Unidades

- HM
- HECB
- HCL
- HPEL
- HRSAJ

## Estrutura

```plaintext
Css/
Js/
Imgs/
index.html
dashboard.html
```

## Como usar

1. Configure o Firebase
2. Adicione sua configuração em:
   `Js/firebase.js`
3. Abra o projeto em servidor local
